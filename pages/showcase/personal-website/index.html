Developer Overview

 This is a demo of what a personal online protfolio could look like. It is clean, simple, and to the point. The layout is compatable with Desktops, Laptops, Tablets and Cell Phones.


Technologies Used:

  • This demo was built with HTML5 and has the future in mind as far as scalability
  • This demo uses Javascript to control elements on the fly at load time to ensure compatability
  • Styles are provided by W3Schools with their w3.css library.
  • This demo is minimal and only includes the basics of a website, but features can be added.
    Such as:
    • More Pages
    • Social Media links and feeds (twitter)
    • Animated Banner Image
